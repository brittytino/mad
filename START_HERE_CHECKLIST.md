# Start Here Checklist (Exam Day)

1. Create new Empty Views Activity project.
2. Choose one problem folder from this repository.
3. Copy XML into res/layout and Kotlin into your package.
4. Register activities/service in AndroidManifest.xml.
5. Add required permissions and dependency lines.
6. Replace any drawable names used in code/XML.
7. Run once and test with sample input before submission.

## 5-minute rewrite order
- UI layout
- id mapping in Kotlin
- validation
- click listeners
- output (TextView/Toast/Intent)
